OWL alignment resources

A few pages for providing information about efforts towards providing a decent environment for aligning OWL ontologies.

We proposed an alignment format and API which is presented technically below. A general overview can be found here.

An alignment format in RDF/XML.

Ontologies

Evaluators

The Owl-Lite Alignment algorithm home page and the associated VisOn visualizer (University of Montréal)

A servlet built around the API for processing, editing and rendering alignments (University of Innsbruck).
