Alignment API and implementation

Here is the new alignment API and implementation homepage. It is hosted by gforge.inria.fr which helps the developement process (by integrating theses pages within the SVN repository for instance).

We proposed an alignment format and API which is presented technically below. A general overview can be found here.

[VisOn]
The VisOn ontology visualizer now works with any implementation of the API. It provides the opportunity to load and display ontologies, load, run, display and compare alignment algorithms.

Related resources

[The OLA algorithm]
A matching algorithm for OWL-Lite that measures distance between all ontology components and perform an iterative fixpoint search (University of Montréal and INRIA Rhône-Alpes).
[OWL Ontology Aligner]
A server built around the API for processing, editing and rendering alignments (University of Innsbruck).
[Ontology Alignment Evaluation Initiative]
Benchmarks for evaluating ontology matchers and benchmarking campaigns (since 2004).
[ontologymatching.org]
The central resource for ontology matching and alignment information.
